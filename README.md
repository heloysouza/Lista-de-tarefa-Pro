# 📝 Lista de Tarefas Pro

Uma aplicação simples e poderosa de lista de tarefas feita com **HTML, CSS e JavaScript puro**, com foco em produtividade e experiência do usuário.

---

## 🚀 Funcionalidades

* ✅ Adicionar tarefas
* 📂 Categorizar tarefas (Trabalho, Estudo, Pessoal)
* ⏰ Definir prazo com data e hora
* 🔔 Notificações automáticas antes do vencimento
* 🌙 Modo escuro/claro
* 🗂️ Filtro por categorias
* ✔️ Marcar tarefas como concluídas
* ❌ Remover tarefas
* 💾 Persistência com LocalStorage (salva automaticamente)

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais de:

* Manipulação do DOM
* Eventos em JavaScript
* Armazenamento no navegador (LocalStorage)
* Estruturação de interface com HTML e CSS
* Lógica de programação aplicada

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (com variáveis e modo escuro)
* JavaScript (Vanilla JS)
